# Planta_Industrial_Sensores

Este é o repositório dos trabalhos que realizei na matéria de Linguagem de Programação.

A ideia é simular um sistema que controla o cadastro de sensores em uma planta industrial, separados por setores. Os sensores possuem no máximo duas leituras diárias.

## Versões

### Versão 1

Possui os cadastro de setores e sensores, podendo adicionar mais, alterar algumas informações dos setores (dos sensores, não), adicionar leitura, mostra alguns tipos de relatórios e possui duas funções de pesquisa.

### Versão 2

É implementado a partir da primeira versão, podendo agora, deletar setores e sensores, alterar algumas informações dos sensores, houve a troca da utilização de vetores como principal estrutura de dados para a lista encadeada. Será adicionado a utilização de arquivos.
